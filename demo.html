Installation

Option 1: Install .crx File (Easiest)

  1. Download pastas-pomodoro-v1.0.0.crx
  2. Open Chrome and go to chrome://extensions
  3. Enable "Developer mode" (top right)
  4. Drag and drop the .crx file onto the extensions page

Option 2: Download Release

  1. Download pastas-pomodoro-v1.0.0.zip
  2. Extract the zip file
  3. Open Chrome and go to chrome://extensions
  4. Enable "Developer mode" (top right)
  5. Click "Load unpacked" and select the extracted folder

Option 3: Build from Source

  1. Clone or download the repository from GitHub
  2. Run npm install to install dependencies
  3. Run npm run build to create the dist folder
  4. Open Chrome and go to chrome://extensions
  5. Enable "Developer mode" (top right)
  6. Click "Load unpacked" and select the dist/ folder

How It Works

Click the extension icon in your browser toolbar to open the popup. Choose between the Timer tab for productivity tracking or the Transform tab to apply themes to any webpage.

Medieval Theme

Transport yourself to a time of knights and castles. The Medieval theme applies parchment-style backgrounds, serif fonts, and a warm brown color scheme to any webpage you visit. Perfect for reading historical content or adding a touch of elegance to your browsing.

Kawaii Theme

Embrace the cuteness! The Kawaii theme transforms pages with soft pink backgrounds, rounded corners, and playful anime-inspired styling that makes browsing more fun. Ideal for creative work or when you need a mood boost.

Pomodoro Timer

Stay focused with the built-in Pomodoro timer. Set 25-minute work sessions, take breaks, and track your productivity. The timer can be popped out to any webpage and dragged to your preferred position.

Tech Stack

Built with React, TypeScript, and Vite. Uses Chrome Extension Manifest V3 for modern browser compatibility.
