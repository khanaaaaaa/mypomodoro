# Pasta's Pomodoro 

## What it is

Pasta Pomodoro is a chrome web extension I built that helps with productivity and keeping track of the time using a pomodoro based timer system, and can change the themes and font of your page according to your selection and mood. And if you want something random, there's a button for it too!

## Features

- Theme Transformation
- Pomodoro Timer
- Break Management
- Session Control

## Installation

1. Install [Pasta Pomodoro](https://chromewebstore.google.com/detail/pastas-pomodoro/phaelkkehkdfnionlbdjaiemklclgifa)

## OR

1. Download [pastapomodoro.crx](https://github.com/khanaaaaaa/mypomodoro/releases/tag/v1)
2. Open Chrome and go to `chrome://extensions`
3. Enable "Developer mode" (top right)
4. Drag and drop the .crx file onto the extensions page

## Why I made it

When I saw there was a sidequest in Flavortown for Chrome Extensions, I wanted try to build something cool even when I didn't know much about Chrome Extensions nor GitHub. I wanted to make something I would use, and what's more useful for a student than a Pomodoro Timer! (Exams were going on when I made it so maybe a self awarness of procrastination led me to build this.)

## How I made it

RabbitHole of tutorials after tutorials and a LOT of debugging. Used vite which I was a little familiar with but it still was a pain to just to get it to work somewhat at the beginning

## What I struggled with and what I learnt

As this was my first project in GitHub, I learnt most of the basics. Got rejected from shipping in FlavorTown 9 times as I did not understand what .crx was and which link I had to keep as a demo. Was a tough process reshipping and fixing stuff again and again, but was extremely fulfilling when it finally got approved so I learnt a lot of patience from this project too. Though it may be a little imperfect, the journey of making something I actually love was fulfilling and surprisingly fueled more love for coding.
